# تبدیل عکس به متریال (Material Transform)

یک اپ اندروید (Kotlin + Jetpack Compose) که یک عکس می‌گیرد و با استفاده از هوش‌مصنوعی تصویری موضوع اصلی عکس را به یکی از متریال‌های زیر تبدیل می‌کند:

1. 🕯️ شمع
2. 🗿 مجسمه سنگی
3. 🪵 مجسمه چوبی
4. 🤖 اکشن فیگور
5. 🏺 مجسمه دکوری

## کدام سرویس هوش‌مصنوعی و چرا؟

این پروژه از **ModelsLab** و اندپوینت رایگان `realtime/img2img` آن استفاده می‌کند.
سطح رایگان ModelsLab شامل **۱۰۰ درخواست در روز** است و به هیچ کارت بانکی یا
صورت‌حساب نیاز ندارد.

## مراحل راه‌اندازی

### ۱. گرفتن کلید API رایگان
1. به آدرس https://modelslab.com بروید و ثبت‌نام کنید (رایگان، بدون کارت بانکی).
2. وارد داشبورد شوید و از بخش API Key، کلید خودتان را کپی کنید.

### ۲. باز کردن پروژه
1. Android Studio را باز کنید، گزینه‌ی **Open** را بزنید و پوشه‌ی
   `MaterialTransformApp` را انتخاب کنید.
2. صبر کنید تا Gradle Sync تمام شود.

یا از GitHub Actions برای ساخت خودکار APK بدون نیاز به Android Studio استفاده کنید
(فایل `.github/workflows/build.yml` از قبل در پروژه موجود است).

### ۳. اجرای اپ
1. اپ را روی گوشی نصب کنید.
2. کلید API را در فیلد بالای صفحه («کلید API رایگان ModelsLab») وارد کنید.
3. یک عکس انتخاب کنید، یکی از پنج متریال را انتخاب کنید و «تبدیل کن» را بزنید.
4. نتیجه را با دکمه‌ی «ذخیره در گالری» ذخیره کنید.

## ساختار کد

- `MaterialType.kt` — پنج گزینه‌ی متریال، همراه با متن دستور (prompt) مخصوص هرکدام.
- `ModelsLabApiClient.kt` — کلاینت شبکه (OkHttp) که عکس ورودی را به‌صورت Base64
  همراه با prompt به ModelsLab می‌فرستد و تصویر خروجی را دانلود می‌کند.
- `MainActivity.kt` — رابط کاربری Compose: انتخاب عکس، انتخاب متریال، دکمه‌ی تبدیل،
  نمایش نتیجه و ذخیره در گالری.

## نکات مهم

- کلید API را در گیت‌هاب عمومی/کد به‌اشتراک‌گذاشته‌شده قرار ندهید.
- سطح رایگان روزانه ۱۰۰ درخواست است؛ بعد از اتمام سهمیه، فردا دوباره ریست می‌شود.
- مستندات رسمی: https://docs.modelslab.com/image-generation/realtime-stable-diffusion/image-to-image
